PayForBlob

The site is designed to send a regular transaction and receive a PayForBlob transaction in return via the Keplr wallet.

Here's a quick guide on how to use the site:

  1. Install and configure the Keplr extension:
  2. Connect your Keplr wallet: Click the "Connect" button on the site to connect your Keplr wallet to the site. The Keplr window will ask for permission to access your wallet information. After connecting, your address will be displayed on the site.
  3. Send the transaction from TIA to the already registered address and the fixed amount of 0.0025 : You only need to enter data such as Namespace ID for example "0c204d39600fddd3" and Message in the form on the site. This data is used to send the request to the server.
  4. Click the "Send" button to send the transaction. The site will check your balance and try to send the transaction through your Keplr wallet.
  5. After successfully sending the transaction, you will see the results as TxHashTransfer and TxHashPFB. These transaction hashes can be used to track the status of transactions on the network.

If you encounter any problems while using the site, please contact the site developer using the contact information provided on the site.
